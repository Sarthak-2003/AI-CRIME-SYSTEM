# AI-CRIME-SYSTEM
AI-powered Crime Investigation System using Java &amp; Python that detects IPC/CRPC sections, generates Hindi voice reports and creates automated video case summaries.
🇮🇳 AI Crime Investigation System

      An end-to-end AI-powered Crime Analysis and Investigation platform built using Java (Swing GUI) and Python (AI Engine + Law Mapping + Media Generation).
      This system intelligently analyzes crime descriptions and automatically determines:
      Applicable IPC Sections
      Applicable CRPC Sections
      Punishment Details
      Hindi Audio Case Report
      MP4 Video Case Summary   
      Timestamped Case Archive

🚀 Key Highlights

  🔎 Advanced Fuzzy Matching (Handles spelling errors like “murdur” → “murder”)
  ⚖️ Automated IPC & CRPC Section Detection
  🎤 Hindi Female Voice Report Generation
  🎬 Automatic MP4 Video Case Creation
  📁 Structured Case Folder Management
  🖥️ Interactive Java Swing GUI
  🔄 Smooth Auto-Scrolling Investigation UI
  📊 Expandable Analytics Architecture
  🧠 System Architecture
